# Integer-Schemes
A testing space for several different integer schemes as they relate to physics problems.
Includes an implementation of the Maxwell-Chern-Simons formulation in 2D and in 3D.
The 2D case includes standard fp64 and the ozaki scheme, while the 3D case only includes the ozaki scheme
You may need to install the specific util packages with python -m p install -e . while in your venv
